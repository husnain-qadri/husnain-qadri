### About Me

I build tools at the intersection of AI and software engineering — from real-time collaborative editors to LLM-powered reading assistants. I'm particularly interested in making language models more reliable and understanding how architectural choices (MoE vs. dense transformers) affect real-world performance.

- 🔬 Researching **hallucination detection** and **adaptive RAG** for more trustworthy LLMs
- 🏗️ Building full-stack apps with **React**, **TypeScript**, **Python**, and **Flask**
- 📊 Running large-scale **LLM benchmarks** (70+ A100 GPU hours across 6 models)
- 🎮 Occasionally building games in Python for fun
